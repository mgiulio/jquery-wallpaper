jQuery Wallpaper: a jQuery Plugin for Full Page Background Images and Slideshows

About

This jQuery plugin will display a full page background image that resize itself to adapt to the window size.

It is also possible to use several images and to cycle between them with a simple slideshow.

Demo

Try to resize the browser window now.
Click on the top link to toggle the visibility of the page foreground.

Usage

Here is anexample from this page. Here we are using plugin default options: 

$.wallpaper([
	'test-images/amsler-grid.gif',
	'test-images/food___by_laprovocation.jpg',
	'test-images/Empire_City_by_gamefan84.jpg',
	'test-images/toys_by_momoclax.jpg',
	'test-images/escher_grid.jpg',
	'test-images/Miami_City_by_Furiousxr.jpg',
	'test-images/pieces-of-me.jpg',
	'test-images/toys_by_hcube.jpg'
]);

The available options are: 

{ // All time are expressed in milliseconds
	images: [],
	imageDuration: 5000, // How much time an image is displayed(not counting transition time)
	transitionDuration: 7000, // The time the transition takes to complete
	firstImageFadeInDuration: 1000
}
